AssertJ Swing overview

AssertJ Swing is a Java library that provides a fluent interface for writing assertions for Swing applications. Its main goal is to improve test code readability and make maintenance of tests easier.
AssertJ Swing is based on JDK standard types assertions, it requires Java SE 6.0 or later and can be used with either JUnit or TestNG.

AssertJ Swing code and issue tracker are hosted on github.
