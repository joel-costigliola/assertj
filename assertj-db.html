AssertJ-DB assertions

AssertJ-DB provides assertions to test data in a database. It requires Java SE 7.0 or later and can be used with either JUnit or TestNG.

If you need additional assertions, just fill a ticket in AssertJ-DB issue tracker.

AssertJ-DB is hosted on github : https://github.com/joel-costigliola/assertj-db.

Big thanks to  Joel Costigliola  for his welcome, for his help and to have accepted to associate this project with a great project like AssertJ.

Régis Pouiller (AssertJ-DB creator)

DataBase assertions quickstart guide

Suppose that in the database there is this table :

MEMBERS
ID NAME FIRSTNAME SURNAME BIRTHDATE SIZE
1 'Hewson' 'Paul David' 'Bono' 05-10-60 1.75
2 'Evans' 'David Howell' 'The Edge' 08-08-61 1.77
3 'Clayton' 'Adam'   03-13-60 1.78
4 'Mullen' 'Larry'   10-31-61 1.70

To quickly start using DataBase assertions, follow the steps below.

1 - Add assertj-db dependency to your project pom.xml

<dependency>
  <groupId>org.assertj</groupId>
  <artifactId>assertj-db</artifactId>
  <version>1.0.1</version>
  <scope>test</scope>
</dependency>

If you use another dependency tool, please check this page to find the relevant assertj dependency declaration for your tool.

2 - Statically import org.assertj.db.api.Assertions.assertThat

... and use your preferred IDE code completion after assertThat.

Example from TableAssertionExamples.java :

import static org.assertj.db.api.Assertions.assertThat;

import org.assertj.db.type.DateValue;
import org.assertj.db.type.Table;

Table table = new Table(dataSource, "members");

// Check column "name" values
assertThat(table).column("name")
                 .value().isEqualTo("Hewson")
                 .value().isEqualTo("Evans")
                 .value().isEqualTo("Clayton")
                 .value().isEqualTo("Mullen");

// Check row at index 1 (the second row) values
assertThat(table).row(1)
                 .value().isEqualTo(2)
                 .value().isEqualTo("Evans")
                 .value().isEqualTo("David Howell")
                 .value().isEqualTo("The Edge")
                 .value().isEqualTo(DateValue.of(1961, 8, 8))
                 .value().isEqualTo(1.77);

On this simple example you can see many concepts of AssertJ-DB (the concepts are simple but i advice to take time to well know them) :

  • the elements :
    • Table which represents a table in the database
    • Column which represents a column of the table
    • Row which represents a row of the table
    • Value which represents a value in a column or in a row
  • the navigation :
    • on the first check, navigation from the table to the column called "name" (column("name") move the assertion on the column), from this column to the first value (value() first call move to the first value) and after to each value (value() next calls move the next value)
    • on the second check, navigation from the table to the row with index 1 (row(1) move the assertion on the row), from this column to the first value and after to each value (value() have a similar behavior from a row and from a column)
  • DateValue : AssertJ-DB works with Java7. So AssertJ-DB provides way to easily compare values with date, time and date/time.
